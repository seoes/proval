# client

SvelteKit app for Proval. Originally scaffolded with [`sv`](https://github.com/sveltejs/cli).

## Developing

From the monorepo root (after `bun install`):

```sh
bun dev
```

Or from `apps/client`:

```sh
bun run dev
bun run dev -- --open
```

## Building

```sh
bun run build
bun run preview
```

See [SvelteKit adapters](https://svelte.dev/docs/kit/adapters) for deployment targets.
